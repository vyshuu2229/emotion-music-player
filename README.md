# Emotion-Based Music Player 

This is a simple project I built using Python to explore how emotions can be connected with music. The idea is to detect a user’s mood and automatically play songs that match how they feel.

##  What this project does
- Identifies user emotion using basic logic
- Plays music based on the detected mood
- Creates a simple and interactive experience

##  Technologies used
- Python

##  Why I built this
I wanted to learn how programming can be used to create something fun and meaningful. This project helped me understand how logic and user experience can work together.
